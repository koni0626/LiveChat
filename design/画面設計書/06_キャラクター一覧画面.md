# キャラクター一覧画面

## 1. 基本情報
- 画面名: キャラクター一覧画面
- 画面ID: `screen-character-list`
- 優先度: B
- 対象ユーザー: 制作者
- Bootstrap:
  - `Bootstrap 5.3`

## 2. 目的
- 作品で使用するキャラクターを一覧管理する

## 3. 画面遷移
### 遷移元
- 作品編集トップ画面

### 遷移先
- キャラクター編集画面

## 4. レイアウト
### 全体構成
- ヘッダー: 作品名
- サイドバー: 作品内メニュー
- メイン領域: 検索、追加ボタン、カード一覧
- 補助領域: なし

### ブレークポイント方針
- `lg`: カード3列
- `md`: 2列
- `sm`: 1列

## 5. 表示項目
| 項目名 | 内容 | データ元 | 備考 |
|---|---|---|---|
| キャラクター名 | 名称 | `/api/v1/projects/<id>/characters` |  |
| 役割 | 役割 | 同上 |  |
| 一人称 | 一人称 | 同上 |  |
| 口調概要 | `speech_style` | 同上 |  |
| 基準画像 | `base_asset` | 同上 | サムネイル |
| 画像固定ルール有無 | `has_image_rule` | 同上 | バッジ表示 |

## 6. 入力項目
| 項目名 | 型 | 必須 | 初期値 | バリデーション | 備考 |
|---|---|---|---|---|---|
| search | text | 任意 | 空 | なし | 名前 / 役割検索 |

## 7. 操作
| 操作 | トリガー | 成功時 | 失敗時 |
|---|---|---|---|
| 検索 | 入力 | 一覧更新 |  |
| 追加 | ボタン | 編集画面へ |  |
| 編集 | カード押下 | 編集画面へ |  |
| 削除 | ボタン | 一覧更新 | トースト表示 |

## 8. 使用API
| 用途 | メソッド | パス | レスポンス利用箇所 |
|---|---|---|---|
| 一覧取得 | GET | `/api/v1/projects/<project_id>/characters` | 一覧表示 |
| 削除 | DELETE | `/api/v1/characters/<character_id>` | 削除操作 |

## 9. 状態管理
- ローカル state: `characters`, `search`, `loading`
- URL クエリ: `search`
- セッション: 認証セッション
- 保存タイミング: 検索時

## 10. エラー・空状態
- ローディング: card skeleton
- 空状態: キャラクター未登録
- バリデーションエラー: なし
- APIエラー: トースト / アラート

## 11. デザインメモ
- Bootstrap コンポーネント: `card`, `badge`, `input-group`, `btn`
- アイコン: 人物アイコン
- トーン: キャラを並べて管理しやすい
