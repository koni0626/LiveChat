# エクスポート画面

## 1. 基本情報
- 画面名: エクスポート画面
- 画面ID: `screen-export`
- 優先度: B
- 対象ユーザー: 制作者
- Bootstrap:
  - `Bootstrap 5.3`

## 2. 目的
- 作品を外部利用可能な形式で出力する

## 3. 画面遷移
### 遷移元
- 作品編集トップ画面

### 遷移先
- 作品編集トップ画面

## 4. レイアウト
### 全体構成
- ヘッダー: 作品名
- サイドバー: 作品内メニュー
- メイン領域: 出力形式選択、履歴
- 補助領域: 出力注意事項

### ブレークポイント方針
- `lg`: 2カラム
- `md`: 1カラム
- `sm`: 1カラム

## 5. 表示項目
| 項目名 | 内容 | データ元 | 備考 |
|---|---|---|---|
| 出力形式 | JSON / HTML / text / image | 固定 | MVPは一部可 |
| 履歴 | export jobs | `/api/v1/projects/<id>/exports` |  |
| ステータス | queued / running / success / failed | 同上 |  |

## 6. 入力項目
| 項目名 | 型 | 必須 | 初期値 | バリデーション | 備考 |
|---|---|---|---|---|---|
| export_type | select | 必須 | 空 | 空禁止 |  |
| options | form/json | 任意 | 空 | なし |  |

## 7. 操作
| 操作 | トリガー | 成功時 | 失敗時 |
|---|---|---|---|
| エクスポート実行 | ボタン | ジョブ追加 | エラー表示 |
| 履歴更新 | ボタン | 一覧更新 | エラー表示 |

## 8. 使用API
| 用途 | メソッド | パス | レスポンス利用箇所 |
|---|---|---|---|
| 一覧取得 | GET | `/api/v1/projects/<project_id>/exports` | 履歴表示 |
| 実行 | POST | `/api/v1/projects/<project_id>/exports` | ジョブ起票 |
| 詳細 | GET | `/api/v1/exports/<export_job_id>` | 結果確認 |

## 9. 状態管理
- ローカル state: exportType、options、jobs
- URL クエリ: なし
- セッション: 認証セッション
- 保存タイミング: 実行時

## 10. エラー・空状態
- ローディング: 履歴 skeleton
- 空状態: 履歴なし
- バリデーションエラー: フォーム表示
- APIエラー: アラート

## 11. デザインメモ
- Bootstrap コンポーネント: `table`, `form-select`, `btn`, `badge`
- アイコン: 出力、ダウンロード
- トーン: 処理状況が分かりやすい
