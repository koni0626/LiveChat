# 作品編集トップ画面

## 1. 基本情報
- 画面名: 作品編集トップ画面
- 画面ID: `screen-project-home`
- 優先度: A
- 対象ユーザー: 制作者
- Bootstrap:
  - `Bootstrap 5.3`

## 2. 目的
- 1作品に関する各編集作業へのハブを提供する

## 3. 画面遷移
### 遷移元
- 作品一覧画面

### 遷移先
- キャラクター一覧 / 編集
- 世界観設定
- ストーリー骨子設定
- シーン一覧・編集
- プレビュー
- エクスポート

## 4. レイアウト
### 全体構成
- ヘッダー: 作品名表示
- サイドバー: 作品内メニュー
- メイン領域: 概要、統計、最近更新シーン
- 補助領域: 推奨アクション

### ブレークポイント方針
- `lg`: 3分割
- `md`: 2段組
- `sm`: 縦積み

## 5. 表示項目
| 項目名 | 内容 | データ元 | 備考 |
|---|---|---|---|
| 作品概要 | タイトル、概要、状態 | `/api/v1/projects/<id>/overview` |  |
| 統計 | キャラ数、シーン数、分岐数、未生成画像数 | `/api/v1/projects/<id>/overview` | カード表示 |
| 最近更新シーン | シーン一覧 | `/api/v1/projects/<id>/overview` | 最大5件 |

## 6. 入力項目
| 項目名 | 型 | 必須 | 初期値 | バリデーション | 備考 |
|---|---|---|---|---|---|
| なし |  |  |  |  |  |

## 7. 操作
| 操作 | トリガー | 成功時 | 失敗時 |
|---|---|---|---|
| キャラクターへ | メニュー | 画面遷移 |  |
| シーン編集へ | ボタン | 画面遷移 |  |
| 最近更新シーンを開く | リスト押下 | シーン編集へ |  |

## 8. 使用API
| 用途 | メソッド | パス | レスポンス利用箇所 |
|---|---|---|---|
| 概要取得 | GET | `/api/v1/projects/<project_id>/overview` | 全体表示 |

## 9. 状態管理
- ローカル state: `overview`, `loading`, `errorMessage`
- URL クエリ: なし
- セッション: 認証セッション
- 保存タイミング: なし

## 10. エラー・空状態
- ローディング: 概要カード skeleton
- 空状態: 最近更新シーンなし
- バリデーションエラー: なし
- APIエラー: アラート表示

## 11. デザインメモ
- Bootstrap コンポーネント: `nav`, `card`, `list-group`, `badge`
- アイコン: 各メニューアイコン
- トーン: 制作の進捗が見える画面
