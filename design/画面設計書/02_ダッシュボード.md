# ダッシュボード

## 1. 基本情報
- 画面名: ダッシュボード
- 画面ID: `screen-dashboard`
- 優先度: A
- 対象ユーザー: 制作者
- Bootstrap:
  - `Bootstrap 5.3`

## 2. 目的
- 作品制作の起点となるホーム画面を提供する
- 最近の作品や主要導線に素早くアクセスさせる

## 3. 画面遷移
### 遷移元
- ログイン画面

### 遷移先
- 作品一覧画面
- 作品新規作成画面
- 作品編集トップ画面
- 設定画面

## 4. レイアウト
### 全体構成
- ヘッダー: サービス名、ユーザー表示
- サイドバー: 主要メニュー
- メイン領域: 最近の作品、通知、利用状況
- 補助領域: クイックアクション

### ブレークポイント方針
- `lg`: 2-3カラムカード配置
- `md`: 2カラム
- `sm`: 1カラム

## 5. 表示項目
| 項目名 | 内容 | データ元 | 備考 |
|---|---|---|---|
| 最近の作品 | 作品カード一覧 | API | 更新順 |
| 新規作品作成 | CTAボタン | 固定 | 上部固定 |
| 利用状況 | 画像生成やAPI利用状況 | 仮 / API | MVPでは簡易でも可 |
| 通知 | システム通知 | 仮 | MVPはプレースホルダ可 |

## 6. 入力項目
| 項目名 | 型 | 必須 | 初期値 | バリデーション | 備考 |
|---|---|---|---|---|---|
| なし |  |  |  |  |  |

## 7. 操作
| 操作 | トリガー | 成功時 | 失敗時 |
|---|---|---|---|
| 作品を開く | カード押下 | 作品編集トップへ遷移 | エラートースト |
| 新規作成へ | ボタン押下 | 作品新規作成へ遷移 |  |

## 8. 使用API
| 用途 | メソッド | パス | レスポンス利用箇所 |
|---|---|---|---|
| セッション確認 | GET | `/api/v1/auth/me` | ユーザー表示 |
| 作品一覧 | GET | `/api/v1/projects` | 最近の作品 |

## 9. 状態管理
- ローカル state: `projects`, `loading`, `errorMessage`
- URL クエリ: なし
- セッション: 認証セッション
- 保存タイミング: なし

## 10. エラー・空状態
- ローディング: skeleton card
- 空状態: 作品未作成メッセージと作成ボタン
- バリデーションエラー: なし
- APIエラー: 通知表示

## 11. デザインメモ
- Bootstrap コンポーネント: `navbar`, `offcanvas`, `card`, `list-group`
- アイコン: ナビゲーション用アイコン
- トーン: 制作開始の勢いが出るUI
