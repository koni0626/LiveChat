# 世界観設定画面

## 1. 基本情報
- 画面名: 世界観設定画面
- 画面ID: `screen-world-edit`
- 優先度: A
- 対象ユーザー: 制作者
- Bootstrap:
  - `Bootstrap 5.3`

## 2. 目的
- 世界観、ルール、禁止事項、固有名詞辞書を編集する

## 3. 画面遷移
### 遷移元
- 作品編集トップ画面

### 遷移先
- 作品編集トップ画面
- シーン一覧・編集画面

## 4. レイアウト
### 全体構成
- ヘッダー: 作品名
- サイドバー: 作品内メニュー
- メイン領域: 左 基本設定 / 中央 詳細設定 / 右 固有名詞辞書
- 補助領域: 保存状況

### ブレークポイント方針
- `lg`: 3カラム
- `md`: 2カラム
- `sm`: 1カラム

## 5. 表示項目
| 項目名 | 内容 | データ元 | 備考 |
|---|---|---|---|
| 世界名 | `world.name` | `/api/v1/projects/<id>/world-context` |  |
| 時代背景 | `era_description` | 同上 |  |
| 場所説明 | `overview` | 同上 |  |
| 技術水準 | `technology_level` | 同上 |  |
| 社会構造 | `social_structure` | 同上 |  |
| 禁止設定 | `forbidden_json` | 同上 |  |
| 固有名詞辞書 | glossary_terms | 同上 |  |

## 6. 入力項目
| 項目名 | 型 | 必須 | 初期値 | バリデーション | 備考 |
|---|---|---|---|---|---|
| world_name | text | 必須 | API値 | 空禁止 | API別名対応 |
| time_period | textarea | 任意 | API値 | なし |  |
| place_description | textarea | 任意 | API値 | なし |  |
| technology_level | textarea | 任意 | API値 | なし |  |
| social_structure | textarea | 任意 | API値 | なし |  |
| world_tone | textarea | 任意 | API値 | なし |  |
| important_facilities | textarea/json | 任意 | API値 | JSONまたは文字列 |  |
| forbidden_settings | textarea/json | 任意 | API値 | JSONまたは文字列 |  |

## 7. 操作
| 操作 | トリガー | 成功時 | 失敗時 |
|---|---|---|---|
| 保存 | ボタン | 保存トースト | エラー表示 |
| AI要約生成 | ボタン | 下書き反映 | エラー表示 |
| 用語追加 | ボタン | 用語モーダル表示 | エラー表示 |

## 8. 使用API
| 用途 | メソッド | パス | レスポンス利用箇所 |
|---|---|---|---|
| 世界観取得 | GET | `/api/v1/projects/<project_id>/world-context` | 初期表示 |
| 世界観更新 | PUT | `/api/v1/projects/<project_id>/world` | 保存 |
| 用語追加 | POST | `/api/v1/projects/<project_id>/glossary` | 辞書編集 |

## 9. 状態管理
- ローカル state: world form、glossary、saving
- URL クエリ: なし
- セッション: 認証セッション
- 保存タイミング: 明示保存

## 10. エラー・空状態
- ローディング: セクション skeleton
- 空状態: glossary 未登録
- バリデーションエラー: フィールド表示
- APIエラー: アラート / トースト

## 11. デザインメモ
- Bootstrap コンポーネント: `card`, `accordion`, `table`, `modal`
- アイコン: 世界、用語集
- トーン: ルールを整理しやすい編集画面
