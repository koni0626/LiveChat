# ストーリー骨子設定画面

## 1. 基本情報
- 画面名: ストーリー骨子設定画面
- 画面ID: `screen-story-outline`
- 優先度: B
- 対象ユーザー: 制作者
- Bootstrap:
  - `Bootstrap 5.3`

## 2. 目的
- 作品全体の骨子、章構成、分岐方針を定義する

## 3. 画面遷移
### 遷移元
- 作品編集トップ画面

### 遷移先
- シーン一覧・編集画面

## 4. レイアウト
### 全体構成
- ヘッダー: 作品名
- サイドバー: 作品内メニュー
- メイン領域: 左 全体設定 / 中央 章一覧 / 右 補足情報
- 補助領域: AI生成導線

### ブレークポイント方針
- `lg`: 3カラム
- `md`: 2カラム
- `sm`: 1カラム

## 5. 表示項目
| 項目名 | 内容 | データ元 | 備考 |
|---|---|---|---|
| 主目的 | main_goal | `/api/v1/projects/<id>/story-outline` |  |
| 主人公の立場 | protagonist_position | 同上 |  |
| 導入 / 骨子本文 | premise / outline_text | 同上 |  |
| 分岐方針 | branching_policy | 同上 |  |
| エンディング方針 | ending_policy | 同上 |  |
| 章一覧 | chapters | `/api/v1/projects/<id>/chapters` |  |

## 6. 入力項目
| 項目名 | 型 | 必須 | 初期値 | バリデーション | 備考 |
|---|---|---|---|---|---|
| premise | textarea | 任意 | API値 | なし |  |
| protagonist_position | textarea | 任意 | API値 | なし |  |
| main_goal | textarea | 任意 | API値 | なし |  |
| branching_policy | textarea | 任意 | API値 | なし |  |
| ending_policy | textarea | 任意 | API値 | なし |  |
| outline_text | textarea | 任意 | API値 | なし |  |

## 7. 操作
| 操作 | トリガー | 成功時 | 失敗時 |
|---|---|---|---|
| 保存 | ボタン | 保存トースト | エラー表示 |
| AI骨子生成 | ボタン | 生成ジョブ起票 | エラー表示 |
| 章追加 | ボタン | モーダル表示 | エラー表示 |

## 8. 使用API
| 用途 | メソッド | パス | レスポンス利用箇所 |
|---|---|---|---|
| 骨子取得 | GET | `/api/v1/projects/<project_id>/story-outline` | 初期表示 |
| 骨子更新 | PUT | `/api/v1/projects/<project_id>/story-outline` | 保存 |
| 骨子生成 | POST | `/api/v1/projects/<project_id>/story-outline/generate` | 生成 |
| 章一覧 | GET | `/api/v1/projects/<project_id>/chapters` | 章表示 |

## 9. 状態管理
- ローカル state: outline form、chapters、job state
- URL クエリ: なし
- セッション: 認証セッション
- 保存タイミング: 明示保存

## 10. エラー・空状態
- ローディング: skeleton
- 空状態: 章未登録
- バリデーションエラー: 各フィールドに表示
- APIエラー: アラート

## 11. デザインメモ
- Bootstrap コンポーネント: `accordion`, `textarea`, `list-group`, `modal`
- アイコン: 章、骨子、AI
- トーン: 全体設計を俯瞰しやすい
