# 画像生成確認画面

## 1. 基本情報
- 画面名: 画像生成確認画面
- 画面ID: `screen-image-review`
- 優先度: A
- 対象ユーザー: 制作者
- Bootstrap:
  - `Bootstrap 5.3`

## 2. 目的
- シーンに対応する生成画像を確認し、再生成や採用を行う

## 3. 画面遷移
### 遷移元
- シーン一覧・編集画面

### 遷移先
- シーン一覧・編集画面
- プレビュー画面

## 4. レイアウト
### 全体構成
- ヘッダー: 対象シーン情報
- サイドバー: なし
- メイン領域: 左 シーン情報 / 中央 画像プレビュー / 右 生成条件
- 補助領域: 再生成履歴

### ブレークポイント方針
- `lg`: 3カラム
- `md`: 2カラム
- `sm`: タブ切替

## 5. 表示項目
| 項目名 | 内容 | データ元 | 備考 |
|---|---|---|---|
| シーン要約 | summary | `/api/v1/scenes/<id>/image-context` | 左 |
| 本文 | narration_text | 同上 | 左 |
| 登場キャラクター | characters | 同上 | 左 |
| 感情 / 場所 / 時間帯 | image_generation | 同上 | 左 |
| 画像 | selected image / generated image | 同上 | 中央 |
| 条件パネル | quality, size, prompt_preview | 同上 | 右 |

## 6. 入力項目
| 項目名 | 型 | 必須 | 初期値 | バリデーション | 備考 |
|---|---|---|---|---|---|
| quality | select | 任意 | `low` | 値制限 |  |
| size | select | 任意 | `1024x1024` | 値制限 |  |
| camera | text | 任意 | 空 | なし | UI用 |
| expression | text | 任意 | 空 | なし | UI用 |
| pose | text | 任意 | 空 | なし | UI用 |
| background_weight | range/select | 任意 | 初期値 | 値制限 | UI用 |
| character_lock_strength | range/select | 任意 | 初期値 | 値制限 | UI用 |

## 7. 操作
| 操作 | トリガー | 成功時 | 失敗時 |
|---|---|---|---|
| 再生成 | ボタン | 画像更新 | エラー表示 |
| 背景のみ再生成 | ボタン | 画像更新 | エラー表示 |
| キャラのみ再生成 | ボタン | 画像更新 | エラー表示 |
| 採用 | ボタン | 選択画像更新 | エラー表示 |
| 不採用 | ボタン | 履歴へ戻す |  |

## 8. 使用API
| 用途 | メソッド | パス | レスポンス利用箇所 |
|---|---|---|---|
| 画面初期表示 | GET | `/api/v1/scenes/<scene_id>/image-context` | 初期表示 |
| 画像一覧 | GET | `/api/v1/scenes/<scene_id>/images` | 履歴 |
| 生成 | POST | `/api/v1/scenes/<scene_id>/images/generate` | 新規生成 |
| 再生成 | POST | `/api/v1/scene-images/<scene_image_id>/regenerate` | 再生成 |
| 採用 | POST | `/api/v1/scene-images/<scene_image_id>/select` | 採用 |

## 9. 状態管理
- ローカル state: context、images、selectedImageId、generationForm
- URL クエリ: `scene_id`
- セッション: 認証セッション
- 保存タイミング: 採用時

## 10. エラー・空状態
- ローディング: 画像プレースホルダ
- 空状態: 画像未生成メッセージ
- バリデーションエラー: 条件パネルに表示
- APIエラー: トースト / アラート

## 11. デザインメモ
- Bootstrap コンポーネント: `card`, `ratio`, `form-select`, `btn-group`, `carousel`
- アイコン: 画像、再生成、採用
- トーン: 画像を主役にした確認画面
