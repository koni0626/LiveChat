# プレビュー画面

## 1. 基本情報
- 画面名: プレビュー画面
- 画面ID: `screen-preview`
- 優先度: A
- 対象ユーザー: 制作者
- Bootstrap:
  - `Bootstrap 5.3`

## 2. 目的
- 制作中の作品をノベルゲーム形式で試遊し、テンポや相性を確認する

## 3. 画面遷移
### 遷移元
- 作品編集トップ画面
- シーン一覧・編集画面

### 遷移先
- シーン一覧・編集画面

## 4. レイアウト
### 全体構成
- ヘッダー: タイトル、章名、戻る
- サイドバー: なし
- メイン領域: 背景 / キャラ表示、本文、選択肢
- 補助領域: ログ、簡易メニュー

### ブレークポイント方針
- `lg`: ゲーム表示を中央固定
- `md`: 幅縮小
- `sm`: テキスト領域優先

## 5. 表示項目
| 項目名 | 内容 | データ元 | 備考 |
|---|---|---|---|
| タイトル | 作品名 | `/api/v1/scenes/<id>/preview` | 上部 |
| 章名 | chapter title | 同上 | 上部 |
| シーン画像 | image | 同上 | 中央 |
| 本文 | narration_text | 同上 | 下部 |
| セリフ | dialogues | 同上 | 下部 |
| 選択肢 | choices | 同上 | 下部 |
| ログ | 表示履歴 | ローカル | MVPは簡易可 |

## 6. 入力項目
| 項目名 | 型 | 必須 | 初期値 | バリデーション | 備考 |
|---|---|---|---|---|---|
| なし |  |  |  |  |  |

## 7. 操作
| 操作 | トリガー | 成功時 | 失敗時 |
|---|---|---|---|
| 次へ | ボタン | 次のテキスト表示 |  |
| 選択肢選択 | ボタン | 次シーンへ | エラー表示 |
| ログ表示 | ボタン | モーダル表示 |  |
| 編集画面へ戻る | ボタン | シーン編集へ遷移 |  |

## 8. 使用API
| 用途 | メソッド | パス | レスポンス利用箇所 |
|---|---|---|---|
| プレビュー初期表示 | GET | `/api/v1/scenes/<scene_id>/preview` | 画面全体 |

## 9. 状態管理
- ローカル state: currentScene、textCursor、logEntries
- URL クエリ: `scene_id`
- セッション: 認証セッション
- 保存タイミング: 任意

## 10. エラー・空状態
- ローディング: 画面中央ローディング
- 空状態: 画像なし時プレースホルダ
- バリデーションエラー: なし
- APIエラー: モーダル or トースト

## 11. デザインメモ
- Bootstrap コンポーネント: `ratio`, `modal`, `btn`, `card`
- アイコン: 再生、ログ、戻る
- トーン: ノベルゲームらしい没入感
