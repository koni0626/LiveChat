# 設定画面

## 1. 基本情報
- 画面名: 設定画面
- 画面ID: `screen-settings`
- 優先度: B
- 対象ユーザー: 制作者
- Bootstrap:
  - `Bootstrap 5.3`

## 2. 目的
- 利用者ごとのUI設定、モデル設定、既定値を変更する

## 3. 画面遷移
### 遷移元
- ダッシュボード
- 作品編集トップ画面

### 遷移先
- ダッシュボード

## 4. レイアウト
### 全体構成
- ヘッダー: 設定
- サイドバー: 全体メニュー
- メイン領域: セクション別設定フォーム
- 補助領域: 説明テキスト

### ブレークポイント方針
- `lg`: 左ナビ + 右フォーム
- `md`: 1カラム
- `sm`: 1カラム

## 5. 表示項目
| 項目名 | 内容 | データ元 | 備考 |
|---|---|---|---|
| 利用モデル設定 | text / select | 将来 | OpenAIモデル設定 |
| デフォルト画像品質 | select | 将来 |  |
| デフォルトサイズ | select | 将来 |  |
| 自動保存間隔 | select | 将来 |  |

## 6. 入力項目
| 項目名 | 型 | 必須 | 初期値 | バリデーション | 備考 |
|---|---|---|---|---|---|
| text_ai_model | text/select | 任意 | 現在値 | なし |  |
| image_ai_model | text/select | 任意 | 現在値 | なし |  |
| default_quality | select | 任意 | `low` | 値制限 |  |
| default_size | select | 任意 | `1024x1024` | 値制限 |  |
| autosave_interval | select | 任意 | 初期値 | 値制限 |  |

## 7. 操作
| 操作 | トリガー | 成功時 | 失敗時 |
|---|---|---|---|
| 保存 | ボタン | 保存トースト | エラー表示 |
| 初期値に戻す | ボタン | 初期値反映 | 確認モーダル |
