# シーン一覧・編集画面

## 1. 基本情報
- 画面名: シーン一覧・編集画面
- 画面ID: `screen-scene-editor`
- 優先度: A
- 対象ユーザー: 制作者
- Bootstrap:
  - `Bootstrap 5.3`

## 2. 目的
- シーンの生成、編集、状態抽出、固定、確認を行う中核画面を提供する

## 3. 画面遷移
### 遷移元
- 作品編集トップ画面
- シーン生成結果確認画面
- 画像生成確認画面

### 遷移先
- シーン生成結果確認画面
- 画像生成確認画面
- プレビュー画面

## 4. レイアウト
### 全体構成
- ヘッダー: 作品名、章名、保存状態
- サイドバー: 章・シーンツリー
- メイン領域: 本文、セリフ、選択肢エディタ
- 補助領域: 現在キャラ、世界観メモ、状態JSON、画像プロンプト

### ブレークポイント方針
- `lg`: 左ツリー / 中央編集 / 右補助の3カラム
- `md`: 左ツリーを offcanvas、中央 + 右の2カラム
- `sm`: 1カラム + タブ切替

## 5. 表示項目
| 項目名 | 内容 | データ元 | 備考 |
|---|---|---|---|
| シーンツリー | 章 / シーン一覧 | `/api/v1/projects/<id>/scenes`, `/chapters` | 左ペイン |
| 本文 | narration_text | `/api/v1/scenes/<id>/editor-context` | 中央 |
| セリフ | dialogue_json | 同上 | 中央 |
| 選択肢 | choices | 同上 | 中央 |
| キャラクター補助 | characters | 同上 | 右ペイン |
| 世界観メモ | world_memo | 同上 | 右ペイン |
| 状態JSON | scene_state_json | 同上 | 右ペイン |
| 画像プロンプト | image_prompt_text | 同上 | 右ペイン |

## 6. 入力項目
| 項目名 | 型 | 必須 | 初期値 | バリデーション | 備考 |
|---|---|---|---|---|---|
| title | text | 任意 | API値 | なし |  |
| summary | textarea | 任意 | API値 | なし |  |
| narration_text | textarea | 任意 | API値 | なし |  |
| dialogue_json | structured | 任意 | API値 | JSON整合 | UIで編集しやすくする |
| choices | structured | 任意 | API値 | 必要に応じて必須 |  |
| note | textarea | 任意 | ローカル | API未保存でも可 |  |

## 7. 操作
| 操作 | トリガー | 成功時 | 失敗時 |
|---|---|---|---|
| 保存 | ボタン | 更新反映 | エラー表示 |
| 次シーン生成 | ボタン | ジョブ起票 | エラー表示 |
| 再生成 | ボタン | ジョブ起票 | エラー表示 |
| 状態JSON抽出 | ボタン | ジョブ起票 | エラー表示 |
| 画像生成 | ボタン | 画像確認画面へ | エラー表示 |
| シーン固定 | ボタン | 固定状態更新 | エラー表示 |

## 8. 使用API
| 用途 | メソッド | パス | レスポンス利用箇所 |
|---|---|---|---|
| 一覧取得 | GET | `/api/v1/projects/<project_id>/scenes` | 左ツリー |
| 詳細取得 | GET | `/api/v1/scenes/<scene_id>/editor-context` | 画面表示 |
| 更新 | PATCH | `/api/v1/scenes/<scene_id>` | 保存 |
| 生成 | POST | `/api/v1/scenes/<scene_id>/generate` | 生成導線 |
| 状態抽出 | POST | `/api/v1/scenes/<scene_id>/extract-state` | 状態更新 |
| 固定 | POST | `/api/v1/scenes/<scene_id>/fix` | 固定更新 |
| 解除 | POST | `/api/v1/scenes/<scene_id>/unfix` | 固定更新 |

## 9. 状態管理
- ローカル state: selectedSceneId、editor form、dirty、jobs
- URL クエリ: `scene_id`, `chapter_id`
- セッション: 認証セッション
- 保存タイミング: 明示保存

## 10. エラー・空状態
- ローディング: ペインごと skeleton
- 空状態: シーン未作成時は作成導線表示
- バリデーションエラー: 入力欄ごとに表示
- APIエラー: 画面上部アラート + トースト

## 11. デザインメモ
- Bootstrap コンポーネント: `offcanvas`, `nav`, `accordion`, `form-control`, `sticky-top`
- アイコン: 生成、固定、保存、プレビュー
- トーン: 作業量が多くても圧迫感を出しすぎない
