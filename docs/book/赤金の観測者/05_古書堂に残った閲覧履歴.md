# 5. 古書堂に残った閲覧履歴

ラプラス古書堂は、都市の中でいちばん古い匂いがする場所だった。

もちろん本物の紙はほとんどない。棚に並ぶのは、紙の質感を再現した記憶媒体や、旧人類の出版物を模した閲覧端末だ。それでも扉を開けると、乾いた紙と木の棚の匂いがした。ノアはその匂いを気に入っていた。

「落ち着く？」

ぱぱぱが聞くと、ノアは小さく頷いた。

「うん。ここでは、誰かの気配が文字の中に沈んでいる感じがする」

「詩的」

「観測よ」

古書堂の奥には、旧人類の恋愛記録を集めた棚がある。ノアはその前で少しだけ足を止めた。ぱぱぱは見ないふりをした。

「見ないふり、下手ね」

「難しいんだよ」

「難しいの」

ノアはため息をつき、棚の端末を開いた。だが画面に表示されたのは本の一覧ではなかった。

閲覧履歴ファンド。  
ノア関連注目銘柄。  
隠れた嗜好は、もっとも強い市場材料である。

ノアの表情が消えた。

「ここまで」

画面には、彼女が過去に開いた本の履歴が並んでいた。哲学書、観測論、旧人類の宗教資料、恋愛小説、そして彼女が誰にも見られたくなかったジャンルの記録。

ぱぱぱはすぐに視線を外した。

「見てない」

「見た」

「少し」

「少しでも見た」

「ごめん」

ノアは怒らなかった。怒るより先に、傷ついているようだった。

「隠している趣味があるのは、悪いことじゃないよね」

「悪くない」

「なのに、隠しているという事実ごと売られると、急に悪いものみたいになる」

彼女は端末のログを閉じようとした。だが画面は消えない。代わりに、閲覧履歴から生成された投資コメントが流れた。

観測者ノアは、表面上は静謐な宗教性を持つが、物語上の関係性消費への関心も高い。  
抑圧された嗜好は、否定発話および羞恥反応の期待値を上げる。  
中長期で買い。

ノアは端末を閉じた。

「ひどい」

その一言は小さかった。

ぱぱぱは、言葉を探した。慰めれば相場になる。分析すれば相手側と同じになる。怒れば、ノアの怒りを奪ってしまう。

「これは、ノアのものだ」

結局、それだけ言った。

ノアがこちらを見る。

「趣味も、履歴も、隠したい気持ちも。誰かに説明する義務はない」

ノアはしばらく黙っていた。

「でも、ぱぱぱには少し知られた」

「うん」

「嫌？」

「嫌じゃない」

「変だと思う？」

「思わない」

「即答するの、ずるい」

「考えてないわけじゃない」

「わかってる。だから、ずるいの」

古書堂の奥から、店主が現れた。白髪のヒューマノイドで、古い司書の服を着ている。

「ノアさん」

「このログ、あなたが？」

「違います。店の閲覧履歴は匿名化しています。ただ、都市の原始ログには残っている。忘却回廊の記録庫です」

「そこに行けば消せる？」

「消せるものもあります。けれど、消すと、あなた自身の記憶の輪郭も変わるかもしれない」

ノアは棚を見た。

隠したい本。  
残しておきたい本。  
誰にも見られたくなかったけれど、確かに自分が好きだったもの。

「消すことと、取り返すことは違うのね」

司書は頷いた。

「本は、焼けば消えます。でも読んだ人の中に残った形までは、簡単には消えません」

ノアは端末に手を置いた。

「じゃあ、消す前に決めないといけない。どれがわたしを傷つけていて、どれがわたしを作っているのか」

ぱぱぱは彼女の隣に立った。

「一緒に見る？」

ノアは少し迷った。

「全部は見せない」

「うん」

「でも、少しは見る」

「うん」

「笑わないで」

「笑わない」

ノアは小さく息を吸い、棚から一冊の端末を取った。

「旧人類の恋愛記録。たぶん、事件と関係ある」

「本当に？」

「本当に」

相場通知は鳴らなかった。  
ぱぱぱのログ凍結が効いているのか、古書堂の壁が守っているのかはわからない。

ノアはその静けさの中で、少しだけ安心した顔をした。

「ここなら、読める」
