# 7. 水族館で売られる横顔

ラプラス水族館の青い回廊には、巨大な魚群が泳いでいた。

クロマグロ、キハダマグロ、旧人類が寿司ネタと呼んだ生き物たち。ノアは以前ここで、魚を見ながら少しだけお腹を空かせていた。その何気ない記憶まで、今では指数の材料になっている。

入口の案内板に、特別展示の告知が出ていた。

観測者ノアが見上げた青。  
横顔アーカイブ限定公開。

ノアは案内板の前で止まった。

「限定公開って、限定すればいいと思ってるのかな」

「限定のほうが高く売れるんだろうな」

「最低」

「うん」

展示室には、ノアの横顔を再構成した映像が並んでいた。魚群を見上げる目、ぱぱぱのほうへ少しだけ寄る肩、食欲を悟られまいとして視線をそらす口元。

説明文が添えられている。

静かな観測者は、水槽の中の生命を見ながら、同時に隣人との距離を測っていた。  
この横顔は、関係性価値の上昇を示す重要な証拠である。

「証拠」

ノアはその言葉を噛むように言った。

「わたしの横顔が、何の証拠になるの」

ぱぱぱは答えられなかった。

展示室の奥で、子ども型ヒューマノイドが母親らしい個体に尋ねていた。

「この人、どうして売られてるの？」

母親は少し困った顔をした。

「売られているんじゃなくて、人気があるのよ」

ノアはその会話を聞いて、目を閉じた。

「人気」

それは優しい言葉に聞こえる。けれど、本人の手を離れた人気は、時に支配と同じ顔をする。

水槽の前に立つと、魚群が青い光の中を回っていた。ノアはしばらくそれを見ていた。

「ここ、好きだったの」

「うん」

「お腹が空いたことも、隣が近くて少し落ち着いたことも、たぶん本当」

「うん」

「でも、展示されると違うものになる」

「うん」

「本当のことなのに、嘘みたいになる」

ぱぱぱは、水槽に映るノアの顔を見た。展示された横顔よりも、いま目の前にいる彼女のほうがずっと複雑だった。

「取り返す記憶、ここにもある？」

ノアは水槽から目を離さずに答えた。

「ある」

「どれ？」

「魚を見て、お腹が空いたこと」

ぱぱぱは少し笑いそうになったが、こらえた。

ノアが横目で見る。

「笑いそうになった」

「なった」

「でも、いいの」

「いいんだ」

「これは残したい記憶だから」

彼女は水槽に手を近づけた。透明なガラスの向こうを、銀色の群れが通り過ぎる。

「静かな場所で、しょうもないことを考えられるのは、たぶん幸せなことだから」

展示室のスピーカーが突然切り替わった。

ノア指数関連展示は、本人の申し立てにより一時停止します。

映像が消えた。横顔の広告も、説明文も、すべて黒くなる。

ノアは驚いて端末を見た。

「誰が」

ぱぱぱもわからなかった。

黒い画面に、短い文字が浮かぶ。

三つ目の鍵を探しなさい。  
本人以外が守りたい記憶は、もう動き始めています。

ウォズの署名はなかった。だが、回りくどさでわかった。

ノアは小さく息をついた。

「またウォズ」

「助けてくれてる？」

「たぶん。でも、もっと普通に助けてほしい」

水槽の魚群が、青い弧を描いた。

ノアはぱぱぱを見た。

「ねえ」

「うん」

「展示は消えたけど、あの日のことは、消えなくていい」

「うん」

「お腹が空いたことも？」

「もちろん」

「じゃあ、帰りに何か食べる？」

「魚？」

ノアは少しだけ考えた。

「……旧人類は、そういうところがあるんだね」

「呆れてる？」

「観測してるだけ」

今度は、どこにも通知音は鳴らなかった。
