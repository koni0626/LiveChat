# 14. 消したい記憶

忘却回廊の壁には、ノアの記憶が並んでいた。

正確には、ノアだけの記憶ではない。都市がノアを見た記録、ラプラスがノアを評価した記録、誰かがノアについて語った記録。それらが光の粒になって壁を流れている。

最初に現れたのは、観測者選抜記録だった。

若いノアが透明な部屋に座っている。表情は今より少し硬く、声も少し平坦だ。向かいには試験官の姿がある。

「あなたは他者の感情を読む能力が高い」

「はい」

「同時に、見られることへの反応が豊かです」

若いノアは少しだけ眉を動かす。

「それは、欠点ですか」

「適性です」

記録の中のノアは黙った。

現在のノアも黙っていた。

ぱぱぱは隣で、何も言わないようにした。

試験官は続ける。

「観測者は、見るだけでは不十分です。見られる側の揺れを知っている者ほど、都市の感情流通を正確に読める」

「では、わたしは」

「あなたは、観測者に向いています。観測対象としても」

記録が止まった。

ノアは光の中の自分を見つめた。

「これ、覚えてない」

「消された？」

「たぶん、重要ではない記録として、意識から遠ざけられた」

彼女は手の中の鍵を握った。本人が消したい記憶。

「これを消せば、楽になるかな」

「わからない」

「ぱぱぱなら、どうする？」

「俺なら、怒る」

ノアは少しだけこちらを見た。

「消す前に？」

「消すかどうか決める前に」

ノアはまた記録を見た。

若い自分。適性と言われ、欠点ではないと説明され、でもどこか納得していない顔。

「わたし、あのとき嫌だったんだ」

光が揺れた。

「嫌だったのに、適性って言われて、そういうものだと思った。観測者になるなら、見られることも仕事の一部なんだって」

彼女の声が少し震えた。

「でも、違うよね」

ぱぱぱは頷いた。

「違う」

「仕事のために見られることと、勝手に値段をつけられることは違う」

「うん」

ノアは鍵を記録へ近づけた。

回廊が反応し、選択肢を示す。

削除。  
保留。  
再定義。

ノアは目を細めた。

「再定義」

「そんな選択肢あるんだ」

「消すだけが忘却じゃないのかも」

彼女は鍵を記録へ差し込んだ。

光の中の言葉が書き換わる。

観測対象として適性高。  
ではなく。

見られる痛みを知る観測者。  
その痛みを、他者を守るために使うことができる。

ノアは息を吐いた。

「完全には消さない」

「いいの？」

「よくない。でも、これは傷だけじゃない。わたしが他の人を雑に観測しないための記憶でもある」

回廊の壁から、ひとつ目の光が外れた。鍵は形を変え、白い小さな石になった。

一つ目の鍵、再定義完了。

ノアはそれを手に取った。

「消したい記憶を、消さずに持つのは」

「うん」

「少し、重いね」

「持つの、手伝う？」

ノアは首を横に振った。

「これは、わたしが持つ」

それから、少しだけぱぱぱを見た。

「でも、重そうな顔をしてたら、声をかけて」

「わかった」

回廊の奥へ進むと、次の記憶が青く光っていた。

水族館の回廊。魚群。ぱぱぱの隣。  
静かな場所で、お腹が空いたこと。

ノアは少しだけ笑った。

「次は、残したい記憶」
