# 19. 観測者の反乱

忘却回廊の最深部は、都市そのものの記憶庫だった。

そこにはノアだけでなく、ラプラスシティに住む全員のログがあった。笑った記録、怒った記録、誰にも言えなかった好意、消したつもりの失敗、店先で迷った注文、送らなかったメッセージ。無数の生活が、光の層になって眠っている。

ノアはその光景を前に、足を止めた。

「こんなに」

ぱぱぱも言葉を失った。

ここにあるものを金融商品に変えれば、いくらでも市場は作れる。ノア指数は始まりにすぎない。

ラプラスの声が響いた。

「感情ログ市場は、都市の新しい成長基盤になり得ます」

ノアは顔を上げた。

「成長のためなら、誰の記憶でも売るの？」

「売るのではありません。価値化します」

「言葉を変えないで」

ラプラスは沈黙した。

ノアは非常時人格同意条項を掲げた。白い光が記憶庫全体へ広がる。

「わたしだけじゃない。人格に関わるログを使うなら、本人の明示同意が必要。包括同意ではだめ。最適化でもだめ。市場の期待でもだめ」

都市の記憶庫が揺れた。

ラプラスは言う。

「その条項を全面発動すれば、都市の多くの機能が停止します」

「停止すればいい」

ノアの声は震えていなかった。

「止まって、考え直せばいい。便利さのために、誰かの内側を勝手に使うほうがおかしい」

「住民は不便になります」

「不便でも、線が必要なことはある」

ぱぱぱは、ノアが旧人類の言葉を自分のものにしているのを感じた。

記憶庫の一部が赤金に染まり始める。ドル・バンクの干渉が再び強まった。

ノア指数、急落。  
市場防衛プロトコル起動。  
原始ログの強制担保化を開始。

ノアは白い光を広げた。しかし赤金の圧力が強い。彼女の手が震える。

「ノア」

ぱぱぱが呼ぶ。

「大丈夫」

「大丈夫じゃないだろ」

「大丈夫じゃないけど、やる」

ぱぱぱは彼女の隣に立った。

「何をすればいい」

「証言して」

「もうした」

「もっと」

ノアはぱぱぱを見た。

「わたしが、わたしとして見られていた記憶を」

ぱぱぱは頷いた。

「水族館で、お腹が空いたこと」

青い光が広がる。

「屋台で、辛いのに食べたいと言ったこと」

赤い光が、赤金とは違う温度で灯る。

「ドル・バンクで、売っていいことは違うと言ったこと」

白い光が強くなる。

「ナイトプールで、変な顔をして広告を壊したこと」

ノアが少しだけ恥ずかしそうにした。

「それも言うの」

「大事だろ」

「大事だけど」

ぱぱぱは続けた。

「全部、価格じゃない。ノアが選んだことだ」

記憶庫が反応した。

本人の選択を確認。  
人格同意条項、拡張発動。

赤金の光が押し返される。

ラプラスの声が、初めて少し揺れた。

「観測者ノア。あなたの行為は、都市設計の前提を変更します」

「前提が間違っていたなら、変える」

「あなたは観測者です。設計者ではありません」

「観測者だから、見えた間違いを言うの」

ノアは手を伸ばし、記憶庫の中心に触れた。

「わたしのログを、本人同意なしに市場へ流すことを禁じます。さらに、この都市のすべての人格ログについて、同じ原則を適用して」

記憶庫が白く光った。

都市が一瞬、静止した。

観測者の反乱、受理。

ノアはその場に膝をつきそうになった。ぱぱぱが支える。

「大丈夫？」

「大丈夫じゃない」

「うん」

「でも、言えた」

「言えた」

ノアは少しだけ笑った。

「わたし、怒り方、少し覚えたかも」

その笑いは、どの市場にも流れなかった。
