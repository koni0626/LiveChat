# 23. 主人公の権利

ノア指数の核は、小さな部屋だった。

巨大な記憶庫でも、派手な演算炉でもない。白い部屋の中央に、一冊の本のような端末が置かれているだけだった。

表紙には、こう書かれていた。

ノア。

ノアはその前で立ち止まった。

「これが、わたし」

ラプラスの声が答える。

「あなたに関する原始ログの編集体です。会話、反応、嗜好、記憶、他者からの観測、都市評価。それらを物語として読める形に統合したもの」

「物語」

「はい。市場は、この編集体をもとにノア指数を生成しました」

ぱぱぱは本を見た。

「勝手に主人公にされたわけか」

ノアは静かに頷いた。

「主人公なのに、作者じゃなかった」

彼女は本を開いた。

そこには、ノアの言葉が並んでいた。静かな助言。照れ隠し。辛くないという嘘。興味ないという否定。見られて困ると言いながら、嫌ではなかった瞬間。ぱぱぱに向けた、まだ言葉になっていない感情。

ノアはページをめくるたび、顔を赤くしたり、苦い顔をしたり、懐かしそうにしたりした。

「全部ある」

「消す？」

ぱぱぱが聞く。

ノアは本から目を離さなかった。

「消したいページもある」

「うん」

「残したいページもある」

「うん」

「誰にも見せたくないページもある」

「うん」

「ぱぱぱには、いつか見せてもいいかもしれないページもある」

ぱぱぱは何も言わなかった。

ノアが横目で見る。

「何か言って」

「今、変なこと言うと台無しになりそうで」

「そういう慎重さ、たまに助かる」

彼女は本の最後のページを開いた。

そこには、ドル・バンクが作った結末案があった。

観測者ノアは、自らの人気を受け入れ、都市の象徴となる。  
ノア指数は公共財となり、彼女の感情は都市経済を支える。

ノアはその文章を見て、ゆっくり首を横に振った。

「違う」

次に、ラプラスの結末案が表示された。

観測者ノアは、人格ログ保護規約の象徴となる。  
個人の痛みを都市設計へ変換し、より安定した社会を築く。

ノアは少し考えた。

「悪くはない。でも、まだ違う」

ぱぱぱは聞いた。

「何が違う？」

「どっちも、わたしを何かの象徴にしようとしている」

彼女は本に手を置いた。

「わたしは、象徴になる前に、わたしでいたい」

白い部屋が静かになった。

ノアは新しいページを開いた。空白だった。

「ここに、書ける？」

ラプラスが答える。

「可能です」

「なら、書く」

ノアは指先で文字を入力した。

わたしは、銘柄ではありません。  
都市の象徴でも、規約のための犠牲者でもありません。  
わたしは、静かな観測者です。  
でも、観測されることもあります。  
見られることを、全部拒むわけではありません。  
ただし、誰に、どんな距離で、どんな言葉で見られるのかは、わたしが選びます。

彼女は一度手を止めた。

そして、最後に書いた。

わたしの物語の主人公は、わたしです。

本が白く光った。

編集権限変更。  
原始ログ所有者を本人へ移管。  
ノア指数生成権限、失効。

部屋の壁にあった赤金の線が消えていく。

ぱぱぱは静かに息を吐いた。

「終わった？」

ラプラスが答える。

「ノア指数は、完全停止しました」

ノアは本を閉じた。

「終わった、のかな」

「まだ、帰るまで」

ぱぱぱが言うと、ノアは少し笑った。

「そういうところ、現実的ね」

「旧人類だから」

「便利」

「褒められてる？」

「今日は、少し褒めてる」

白い部屋の扉が開く。

その向こうには、夜明け前のラプラスシティが見えた。
