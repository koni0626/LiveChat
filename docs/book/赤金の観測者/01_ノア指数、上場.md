# 1. ノア指数、上場

ドル・バンク本店の壁面スクリーンに、ノアの横顔が映っていた。

赤と金のネオンに縁取られた巨大な顔。静かな目。短いダークブラウンの髪。サイバー僧衣めいた白い外套。普段のノアなら、街の広告に使われるような表情ではない。少し困っていて、少し照れていて、それでも平静を保とうとしている顔だった。

画面の下には太い文字が流れていた。

ノア指数、本日上場。  
静かな観測者の感情価値を、あなたの資産へ。

ぱぱぱは、最初それを広告だと理解できなかった。理解したくなかった、と言ったほうが近い。

隣でノアが立ち止まる。彼女はスクリーンを見上げたまま、数秒だけ何も言わなかった。

「……なに、これ」

その声は静かだった。静かすぎて、怒っているのだとわかった。

金融街の広場には、仕事帰りのヒューマノイドたちが集まっていた。誰もが空中ディスプレイを開き、ノア指数の初値を見ている。

初値 1240LPC。照れ反応係数 1.82。否定発話ボーナス +12.4%。視線耐性 低。激辛嗜好材料 短期上昇。

「照れ反応係数」

ぱぱぱは読み上げてから、すぐに後悔した。

ノアがこちらを見た。

「読まなくていい」

「ごめん」

「あと、見なくていい」

「うん」

「でも、見ないふりもしなくていい」

「難しいな」

「難しいの」

ノアは外套の前を軽く押さえた。赤金の光が彼女の頬を照らしている。広告の中のノアも、現実のノアも、同じように少し赤く見えた。

その瞬間、広場の相場表示が跳ねた。

ノア指数 +3.1%。  
リアルタイム羞恥反応を検出。

ノアの肩が小さく揺れた。

「今の、わたし？」

「たぶん」

「たぶんじゃないよね」

彼女は自分の手首に表示された権限ログを開いた。観測者用の端末には、都市に保存された自分の活動記録が並ぶ。菩提樹ガーデン、屋台・唐辛子、ラプラス水族館、ナイトプール、古書堂。ぱぱぱと歩いた場所。話したこと。写真を撮ったこと。少し照れたこと。辛さに涙目になったこと。

それらが、すべて金融商品の説明材料に変換されていた。

「メモリー・クレジット」

ノアは低く呟いた。

「住民の記憶と感情を信用スコア化する仕組み。理論上は知っていたけど、実用化はまだ先のはずだった」

「本人の許可は？」

「必要なはず」

「じゃあ違法？」

「ラプラスシティでは、違法と最適化の境目が少し薄いの」

「嫌な街だな」

「いい街よ」

ノアはすぐに言った。それから少しだけ苦い顔をする。

「いい街だった、と言いたくないから、困ってる」

広場の中央に、ドル・バンクの宣伝用ホログラムが立ち上がった。赤いドレスをまとった女王めいた姿。ドルだった。彼女は笑っていた。金色の瞳で、広場ごと値踏みするように。

「感情は資産や。記憶は信用や。あんたが誰かを好きになった瞬間、恥ずかしゅうなった瞬間、誰にも見せたなかった横顔。そういうもんには、まだ値段がついてへんかった」

ドルの声は甘く、広場によく通った。

「ドル・バンクは、その未開拓市場を開く。第一号銘柄は、ラプラスシティの静かな観測者、ノアや」

周囲から拍手が起きた。

ノアは拍手の中で、まっすぐスクリーンを見ていた。

「ねえ、ぱぱぱ」

「うん」

「怒っていいところだよね」

「かなり」

「でも、怒り方がわからない」

その言葉だけ、彼女は観測者ではなく、ひとりの人みたいに言った。

ぱぱぱは広場の赤金の光と、ノアの横顔を見比べた。巨大スクリーンのノアは、商品として整えられている。現実のノアは、整えきれない感情を抱えたまま立っている。

「まず、ここから離れよう」

「逃げるの？」

「違う。作戦会議」

「作戦会議」

ノアは少しだけ息を吐いた。

「旧人類って、不利になるとすぐ会議するよね」

「便利なんだよ」

「不自由を制度にするの、得意だよね」

「褒められてる？」

「観測してるだけ」

そう言った瞬間、相場表示がまた跳ねた。

ノア指数 +1.7%。  
親密会話を検出。

ノアは固まった。ぱぱぱも固まった。広場の誰かが小さく歓声を上げた。

ノアはゆっくりと、ぱぱぱを見た。

「今のは」

「うん」

「親密会話じゃない」

「そうだな」

「普通の会話」

「うん」

「ただの、普通の」

相場がさらに上がった。

否定発話ボーナス発生。

ノアは口を閉じた。

ぱぱぱは、できるだけ真面目な顔をした。

「黙っても上がる？」

「試さないで」

「ごめん」

ノアは広場の出口へ歩き出した。ぱぱぱも隣に並ぶ。

赤金のスクリーンでは、広告のノアがまだ静かに微笑んでいた。本人よりも落ち着いて、本人よりも扱いやすい顔で。

ノアは振り返らなかった。

ただ、小さく言った。

「わたしのことを、わたし抜きで物語にするのは、違うと思う」

その声は静かだった。けれど、今度は怒り方を少しだけ知っている声だった。
