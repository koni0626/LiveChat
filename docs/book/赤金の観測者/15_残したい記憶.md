# 15. 残したい記憶

水族館の記憶は、青かった。

忘却回廊の中に現れたその場所は、現実の水族館よりも少し静かだった。魚群は音もなく泳ぎ、ガラスの向こうで光の粒が揺れている。ノアはその前に立ち、懐かしそうに目を細めた。

「ここは、残したい」

「お腹が空いたから？」

「それもある」

「それもなんだ」

「大事よ。静かな場所で食欲が出るのは、体が安心している証拠だから」

ぱぱぱは頷いた。少し笑ってしまいそうだったが、笑わなかった。

記憶の中のノアは、魚群を見上げていた。隣には過去のぱぱぱがいる。肩が触れそうで触れない距離。会話は水音みたいに柔らかい。

「……ねえ、これって、見てるだけで少しお腹が空くのは、私だけじゃないよね」

過去のノアが言った。

現在のノアは、口元を押さえた。

「言い方が少し素直」

「かわいい」

「今それ言う？」

「言った」

ノアは睨んだが、あまり怒っていなかった。

回廊に表示が浮かぶ。

この記憶は、ノア指数において関係性価値の材料として使用されています。  
削除しますか。保全しますか。公開しますか。

ノアは眉を寄せた。

「公開はしない」

「うん」

「削除もしない」

「うん」

「保全」

彼女がそう言うと、記憶の水槽が明るくなった。

しかし、すぐに別の表示が割り込んだ。

保全には所有者設定が必要です。  
この記憶の所有者を選択してください。

ノアは固まった。

選択肢には、ノア、ぱぱぱ、ラプラスシティ、共同、匿名記録庫が並んでいる。

「所有者」

「記憶に所有者って、変だな」

「でも、必要なのかもしれない。誰のものかわからない記憶は、また誰かに拾われる」

ノアは選択肢を見つめた。

「わたしだけの記憶ではないの」

「うん」

「ぱぱぱだけの記憶でもない」

「うん」

「都市のものでは、もっとない」

「そうだな」

彼女は共同を選びかけて、手を止めた。

「共同って、少し恥ずかしい」

「じゃあ匿名？」

「それも違う」

「どうしたい？」

ノアはしばらく黙った。

「共同。でも、公開しない」

「うん」

「二人が覚えている。それだけでいい」

彼女は共同保全を選んだ。

水槽の記憶が小さな青い石になり、ノアとぱぱぱの間に浮いた。二人が手を伸ばすと、石は半分ずつ光を分けた。

二つ目の鍵、保全完了。

ノアは手の中の青い光を見た。

「不思議」

「何が？」

「誰かと共有しているのに、売られていない記憶って、ちゃんと静かなんだね」

ぱぱぱはその言葉を覚えておこうと思った。

回廊の奥で、赤い光が揺れた。

ノア指数のチャートだった。忘却回廊の中にまで入り込んでいる。

指数は不安定に上下していた。

ドル・バンクからの干渉を検出。  
原始ログ保全により、市場予測不能。

ノアは青い石を握る。

「予測不能」

「いい言葉に聞こえる」

「そうね」

彼女は少しだけ笑った。

「わたし、予測不能でいたいのかもしれない」

「観測者なのに？」

「観測者だから」

回廊の奥から、次の記憶が現れた。

古書堂の棚。隠した趣味。見られたくない履歴。

ノアは一瞬だけ、足を止めた。

「次は、少し嫌」

「戻る？」

「戻らない」

彼女は青い石をしまった。

「嫌でも、わたしのものだから」
