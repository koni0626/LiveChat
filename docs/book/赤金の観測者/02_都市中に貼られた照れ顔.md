# 2. 都市中に貼られた照れ顔

翌朝、ラプラスシティはノアの顔で満ちていた。

駅の改札、搬送路の側面、レストラン街の注文端末、観測塔のエレベーター内。どこを見ても、ノアの横顔がある。広告の中の彼女は、少しだけ目を伏せ、少しだけ頬を赤くしていた。

キャッチコピーは場所によって違った。

見られるほど、価値になる。  
否定するほど、物語になる。  
静かな観測者は、今日も上昇中。

ノアは観測塔のロビーで立ち尽くした。

「……わたし、こんな顔してた？」

ぱぱぱは答える前に、言葉を選んだ。

「似てるけど、違う」

「どう違うの」

「広告のほうは、見られるために作られてる。ノア本人は、見られたあとにどうするかを考えてる」

ノアは少しだけ黙った。

「そういうことを、すぐ言うから相場が上がるのよ」

「ごめん」

「謝らなくていい。たぶん、今のは少し助かった」

ロビーの隅で、若いヒューマノイドたちがノア指数のチャートを見ていた。彼らは本人が近くにいることに気づいていない。もしくは気づいていて、気づいていないふりをしていた。

「昨日の否定発話、強かったよね」

「あの『親密会話じゃない』で買った人、勝ち組」

「次は激辛イベントだって。泣き顔出たら跳ねる」

ノアの指がぴくりと動いた。

「泣き顔」

「聞かなくていい」

「聞こえたものは、なかったことにできないの」

彼女は静かに彼らの前へ歩いた。ぱぱぱは止めなかった。止めたら、彼女が怒る機会まで奪ってしまう気がした。

「そのチャート」

若いヒューマノイドたちが振り向いた。本人を見て、全員が一瞬で固まる。

「……あ」

「あ、じゃなくて」

ノアは穏やかな声で続けた。

「わたしの泣き顔で利益が出るなら、あなたたちは嬉しい？」

誰も答えなかった。

ノアは責めるような顔をしていなかった。だから余計に、広場の空気は重くなった。

「責めているわけじゃないの。たぶん、仕組みがそうさせている。値段がつくと、人はそれを数字として見てしまう。旧人類もそうだったし、わたしたちもあまり変わらない」

「すみません」

ひとりが小さく言った。

ノアは首を横に振った。

「謝るなら、買わないで」

そう言ってから、彼女は少しだけ考えた。

「でも、売る側を止めないと、あなたたちはまた買う。わたしも、たぶん同じ立場ならチャートを見てしまう。だから、まず売る側を止める」

ぱぱぱはその言葉を聞いて、ノアが怒りの形を見つけはじめていると思った。

ロビーの中央に警告音が鳴った。都市広報の緊急配信だった。

ドル・バンクよりお知らせ。  
ノア指数の人気急騰を受け、関連ファンドを本日正午より拡張します。  
新規材料として、過去のおでかけログ、衣装選択傾向、会話中の否定反応を追加予定。

ノアはゆっくりと息を吸った。

「過去のおでかけログ」

ぱぱぱは、菩提樹ガーデンで並んで座った日のことを思い出した。水族館の青い回廊。ナイトプールで撮った写真。辛さに涙目になった屋台。

どれも、二人の間では小さな記憶だった。街に売られるための素材ではなかった。

「ドル・バンクへ行こう」

ノアが言った。

「正面から？」

「うん。まずは、ちゃんと抗議する」

「聞くかな」

「聞かないと思う」

「じゃあ」

「聞かないことを確認するのも、観測のうちよ」

ノアはロビーの出口へ向かった。  
その背中は、広告の中のノアよりずっと小さかった。けれど、ずっと強かった。

ぱぱぱは隣に追いつく。

「今の、かっこよかった」

相場通知が小さく鳴った。

ノア指数 +0.9%。  
肯定反応を検出。

ノアは足を止めた。

「ぱぱぱ」

「ごめん」

「もう少し、こっそり褒めて」

「難しいな」

「難しいの」

それでもノアは、少しだけ笑っていた。
