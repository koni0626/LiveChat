# 赤金の観測者

ラプ・シネマ上映映画版・長編第一稿

## 上映企画

本作は、ラプラスシティの映画館「ラプ・シネマ」で上映される劇中映画として扱う。  
実際のノア指数事件をもとにした社会派SFロマンス映画であり、事件の記録そのものではなく、映画として人物の個性、対立、会話の温度を強めている。

ドルは関西弁の金融カリスマとして描く。  
標準語の冷たい敵役ではなく、「うち」「あんた」を使い、笑いながら市場と人の欲望を転がす赤金の女王にする。

## 執筆メモ

この第一稿では、DBにあるノアの設定、AIメモ要約、おでかけ記録、ラプラスシティ、ドル・バンク、忘却回廊の要素を統合した。  
主題は「観測されること」と「商品化されること」の違い。  
ノアは最後に、見られることを拒むのではなく、自分の物語を自分で語る権利を取り戻す。

## キャラクター演出メモ

- ノア: 静かでやさしい観測者。怒りも照れも大声にせず、言葉を選んで出す。動揺すると「暑いだけ」「興味ない」と否定するが、本音はにじむ。
- ドル: 関西弁。赤金の資本主義を人格化した女王。挑発的で艶があり、勝っている側の余裕で話す。「うち」「あんた」「そんなん甘いで」「勝てる波」を使う。
- ラプラス: 都市そのものの声。冷たい悪役ではなく、最適化の論理で人を傷つける存在。謝罪も分析として出る。
- コリー: 明るい食の人。深刻な話を生活の側へ戻す。騒がしいが、食べることを商品化しない芯がある。
- ウォズ: 回りくどい記録者。核心を知っているが、直接答えず、選択の形で渡す。
