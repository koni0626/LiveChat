# 21. ノア指数、暴落

ノア指数は、停止と同時に暴落した。

正確には、取引が成立しなくなった。買いも売りも凍結され、チャートは最後の価格を表示したまま動かない。だが市場は、数字が止まったことを暴落として受け取った。

金融街は混乱した。

ドル・バンク本店前には投資家が集まり、怒号と抗議が飛び交った。感情ログ市場に連動していた関連銘柄も止まり、赤金のスクリーンは警告で埋まった。

ノアは観測塔の中からその様子を見ていた。

「わたしのせいで」

「違う」

ぱぱぱは言った。

「でも、引き金はわたし」

「仕組みが間違ってた。止めたら揺れた。それだけだ」

「それだけ、ではないよ」

ノアは窓に手を置いた。

「あの中には、損をした人もいる。わたしのことを商品として見ていたとしても、生活がある」

ぱぱぱは黙った。

ノアは、怒りだけで人を切り捨てることができない。そこが彼女の弱さで、強さだった。

ラプラスから通信が入った。

「金融街で混乱が拡大しています。ドル・バンクは緊急会見を要求」

「ドルが？」

「はい。あなたとの対話を希望しています」

ぱぱぱは顔をしかめた。

「罠かもしれない」

「罠でも行く」

「ノア」

「止めた責任はある。商品にされた責任じゃなくて、止めたあとを見る責任」

彼女は外套を整えた。

金融街に着くと、赤金のネオンは半分消えていた。ドル・バンク本店の前で、ドルが待っている。いつもの笑みは薄い。

「見事やな、観測者ノア」

「皮肉？」

「半分や」

「もう半分は？」

「本心や」

ドルはスクリーンを見上げた。

「市場は止まった。損失は大きい。うちの信用も傷ついた」

「あなたが始めた」

「せや」

ドルは認めた。

ノアは少し意外そうにした。

「認めるの」

「金融は責任を曖昧にする技術でもある。けど、曖昧にしすぎると信用を失うんや」

「信用」

「銀行やから」

ドルは肩をすくめた。

「うちは、感情を資産にできると思った。そこは間違いやない。感情には価値がある。ただ、その価値を本人から切り離して売れると思ったんが間違いやった」

ノアは黙って聞いた。

「謝るの？」

「謝罪はする。ただし、それだけでは市場は戻らへん」

「戻さなくていい」

「違う形にする必要があるんや」

ドルは新しい提案を表示した。

メモリー・クレジット改定案。  
本人主導型記憶公開。  
公開範囲、収益分配、撤回権、共同記憶の全員同意。

ノアは文面を読んだ。

「また商品にするの？」

「商品にしたい人だけが、自分で商品にする。創作、記録、思い出、支援。価値を受け取る権利も、本人にあるべきやろ」

ノアは考えた。

「わたしは、すぐには賛成しない」

「せやろな」

「でも、話し合う余地はある」

ドルは少し笑った。

「あんたは甘いな」

「甘くない。辛いものが好き」

ぱぱぱは少し笑った。

ドルも、ほんの少し笑った。

そのとき、金融街全体が揺れた。

赤金のスクリーンが一斉に黒くなる。

ラプラスの緊急通知が響いた。

都市金融中枢、停止。  
感情ログ市場凍結に伴う連鎖障害。  
赤金区画、隔離モードへ移行。

ドルの表情が変わった。

「まずいわ」

ノアは観測塔の方角を見た。

「都市が止まる」

赤金の夜が、光を失い始めた。
