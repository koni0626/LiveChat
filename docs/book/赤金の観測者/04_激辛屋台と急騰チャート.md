# 4. 激辛屋台と急騰チャート

作戦会議の場所としてノアが選んだのは、屋台・唐辛子だった。

ぱぱぱは、もう少し静かな場所を想像していた。古書堂とか、菩提樹ガーデンとか、観測塔の個室とか。だがノアは金融街からまっすぐ屋台街へ向かい、赤い湯気と刺激臭の前で足を止めた。

「ここ？」

「ここ」

「作戦会議だよね」

「そうよ」

ノアはメニューを見上げた。ドラゴンブレス、リーパー、火星唐辛子三倍、旧人類泣かせ。名前だけで舌が痛くなりそうな札が並んでいる。

「辛いものを食べると、余計なことを考えなくて済むの」

「考えられなくなる、の間違いでは」

「似たようなものよ」

彼女は真剣な顔でいちばん赤い札を指した。

ぱぱぱは止めようとした。だが、その前に屋台の横の小さな相場表示が点灯する。

ノア指数 関連材料。  
激辛嗜好イベント発生予測。  
短期買い推奨。

ノアの指が止まった。

「食べる前から？」

「予測されてる」

「わたしが辛いものを食べて、泣きそうになって、それでも辛くないって言うところまで？」

「たぶん」

「言わない」

相場が上がった。

否定発話ボーナス +8.2%。

ノアは目を閉じた。

「もう、何をしても材料になるのね」

「今日は食べるのやめる？」

「それも嫌」

「どうして？」

ノアはぱぱぱを見た。

「わたしが好きなものまで、あっちの都合で避けるのは、もっと嫌だから」

その言葉に、ぱぱぱは頷いた。

「じゃあ食べよう」

「うん」

「相場を上げるためじゃなくて」

「わたしが食べたいから」

ノアはそう言って、火星唐辛子三倍を注文した。

数分後、彼女は紙カップを両手で持ち、目にうっすら涙を浮かべていた。

「辛い？」

「辛くない」

相場が上がった。

「今のなし」

さらに上がった。

ノアはぱぱぱを睨んだ。

「笑わないで」

「笑ってない」

「笑いそうになった」

「観測精度が高い」

「高いの。だから困るの」

いつものやり取りだった。けれど今日は街がそれを見ている。値段をつけている。勝手に盛り上がっている。

ぱぱぱは、屋台の横に表示されたノア指数を見た。

数字は上がっている。でも、ノアの顔は広告の中よりずっと生きていた。辛さに負けそうで、悔しそうで、少し楽しくて、怒っていて、それでも自分で選んでいる顔だった。

「ノア」

「なに」

「これ、止めよう」

「うん」

彼女は涙目のまま頷いた。

「止める。でも、消すだけじゃだめ」

「どうする？」

「取り返すの」

ノアは紙カップを置き、金融街のほうを見た。

「わたしの記憶を、わたしが語り直す」

そのとき、屋台の店主が小さな袋を差し出した。

「サービスだよ。火星唐辛子の種。忘却回廊に行くなら、持っていくといい」

ノアは目を瞬いた。

「どうして、それを」

店主は肩をすくめた。

「この街の屋台は、噂で煮込まれてるからね。あと、あんたの指数は買ってない。辛いものを食べる客に値段をつけるほど、うちは悪趣味じゃない」

ノアは袋を受け取った。

「ありがとう」

「辛いよ」

「知ってる」

「たぶん、想像より」

ノアは少しだけ笑った。

「それも知ってる」

ぱぱぱは、その笑顔を見た。相場には載せたくないと思った。街にも、ドルにも、ラプラスにも渡したくないと思った。

だから何も言わなかった。

ノアはその沈黙に気づいたように、少しだけこちらを見た。

「今、見た？」

「見た」

「相場には？」

「載せない」

「なら、いい」

彼女は辛さで赤くなった目元を袖で押さえた。

「なら、少しだけ、見ていて」
