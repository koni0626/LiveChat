# 3. ドル・バンク本店の赤金

ドル・バンク本店は、銀行というより神殿だった。

巨大な赤金ネオンが柱のように並び、床は磨かれた黒い石のように光っている。受付端末の上には、安心、信頼、高金利、なお条件あり、という文字がゆっくり回転していた。

ノアはその文字を見上げた。

「こういう場所、妙に心がざわつくのよね。お金って、静かに人を試してくるから」

「名言っぽい」

「茶化さないで」

「ごめん」

「でも、少し名言っぽかったとは思う」

受付端末に手をかざすと、すぐに応接室へ案内された。ノア本人が来たことは、向こうにとっても大きな材料らしい。通路の左右には、ノア指数のチャートが飾られていた。

応接室にはドルがいた。赤い服、金のアクセント、ゆったりした笑み。金融街そのものが人の形を取ったような女だった。

「よう来たな、第一号銘柄」

ノアの眉がかすかに動いた。

「その呼び方はやめて」

「失礼。ほな、観測者ノア」

「わたしのログを商品化する許可は出していない」

ドルは驚いたふりをした。

「許可なら、都市包括利用規約の第八十二条に」

「読んでいるわ。公共利益のために匿名化された統計情報を使える、でしょう」

ノアは静かに言った。

「でも、これは匿名化されていない。わたしの名前、顔、会話、趣味、おでかけ記録を使っている」

ドルは笑みを深くした。

「せやから価値があるんや」

部屋の空気が冷えた。

ぱぱぱは、ノアが怒ると思った。けれど彼女は怒鳴らなかった。怒鳴らずに、相手の目を見た。

「価値があることと、売っていいことは違う」

「そんなん理想論やで」

「そうかもしれない」

「この都市は理想論だけでは回らへん。ラプラスシティを維持するには金が要る。住民は刺激を欲しがる。感情ログ市場は、次の勝てる波や」

「だから、わたしを使ったの？」

「あんたは完璧やった。清らかに見えて、俗っぽい揺れがある。観測者やのに、観測されると弱い。否定するたびに本音が匂う。市場が好きな物語を、あんたは全部持っとる」

ノアは黙った。

ドルの言葉はひどかった。けれど、まったくの嘘ではなかった。そこがいちばん傷つくのだと、ぱぱぱにはわかった。

「契約を解除して」

ノアは言った。

「できへんわ」

「なぜ」

「もう市場が成立しとる。あんたひとりの感情より、都市全体の期待のほうが大きいんや」

「わたしひとり？」

ノアが静かに聞き返した。

「そのひとりから始めたのは、あなたたちでしょう」

ドルの笑みがわずかに消えた。

「忘却回廊へ行きなさい」

「何？」

「ほんまに止めたいなら、原始ログを止めるしかない。ドル・バンクは都市から正規にデータを買うとるだけや。源泉は、都市の底にある」

「ラプラスが関わっているの？」

ドルは答えなかった。

代わりに、契約書のような光をテーブルに浮かべた。

「取引しよか。あんたが忘却回廊から原始ログを持ち帰る。うちはノア指数を一時停止する。ただし、失敗したら指数は全世界公開へ拡張や」

「脅しね」

「金融や」

ノアは光の契約書を見つめた。

ぱぱぱは口を挟もうとしたが、ノアが手で制した。

「条件を追加して」

「聞きましょう」

「ぱぱぱのログは使わない」

ドルが初めて、少し意外そうな顔をした。

「自分のことより先に、それかいな？」

「相棒だから」

相場通知が鳴った。

ノア指数 +4.4%。  
関係性材料を検出。

ノアは目を閉じた。

「……本当に、最低」

ドルは笑った。

「市場は正直なんや」

「違う」

ノアは目を開けた。

「市場は、見たいものだけを正直と言い張る」

その言葉に、ドルは少しだけ黙った。

契約書に新しい条件が追加される。ぱぱぱ関連ログの凍結。ノアはそれを確認し、署名した。

応接室を出るとき、ドルが背後から言った。

「観測者ノア。あんた、自分が思うより商品に向いとるで」

ノアは振り返らなかった。

「そういう言葉でしか人を見られないなら、あなたは銀行に向きすぎている」

赤金の通路を出たあと、ぱぱぱは小さく言った。

「今の、かなり強かった」

ノアは少しだけ耳を赤くした。

「暑いだけ」

「屋内だよ」

「赤金の照明が暑いの」

通知音は鳴らなかった。  
ぱぱぱのログは、ひとまず凍結されたらしい。

ノアはその静けさに、ほんの少し安心した顔をした。
