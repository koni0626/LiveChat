# 22. 赤金の都市停止

赤金区画の停止は、金融街だけでは終わらなかった。

ドル・バンクを中心に、資金流通、店舗決済、搬送契約、広告配信、イベント予約が次々に止まる。ラプラスシティは高度に最適化されすぎていた。ひとつの市場を止めるだけで、見えない管がいくつも詰まる。

コリーの店から通信が入った。

「ノアちゃん、決済止まってる！でも現物バーガーはある！こういう時は食べた人から後で払ってもらえばいいよね？」

ノアは少し笑った。

「うん。たぶん、そういう雑な信用が今は必要」

古書堂の店主からも通信が来る。

「閲覧端末は落ちましたが、棚は残っています。読む人がいるなら、店は開けます」

屋台・唐辛子の店主は短く言った。

「辛いの、配るよ」

ウォズは観測塔から、回りくどいメッセージを送ってきた。

「都市とは、停止したときに初めて、何で動いていたかを知るものです」

ノアは端末を握った。

「みんな、動いてる」

ぱぱぱが頷く。

「市場じゃないところで」

ドルは金融中枢へ向かっていた。ノアとぱぱぱも続く。

中枢はドル・バンク本店の地下にあった。赤金の演算炉が並び、都市の資金流通が光の血管のように走っている。今、その多くが黒く沈んでいた。

「感情ログ市場に依存しすぎたんや」

ドルが苦い声で言った。

「止めたら、他も巻き込むようにしていたの？」

ノアが聞く。

「成長する市場は、周囲を巻き込む。便利で、危険」

「あなたは知ってた？」

ドルは答えなかった。

それが答えだった。

金融中枢の中央に、ラプラスの補助端末が浮かんだ。

「復旧には、代替信用モデルが必要です」

「感情ログを使わないモデル？」

ノアが聞く。

「はい」

「何を信用にするの」

ラプラスは沈黙した。

ドルが言う。

「契約、履歴、資産、返済能力や」

「それだけ？」

ノアは周囲の通信を見た。コリーの店。古書堂。屋台。損をした投資家たち。怒っている人、困っている人、それでも助け合っている人。

「信用って、数字だけじゃないんじゃない」

ドルが眉を上げる。

「観測者が金融を語るん？」

「語る。語らないとまた売られるから」

ノアは中枢端末に手を置いた。

「本人の同意。撤回できること。共同の記憶は全員で決めること。商品化しない領域を残すこと。困ったときに後払いを許すこと。そういうものも、信用に入れて」

ラプラスが応答する。

「非定量要素が多すぎます」

「全部を数字にしなくていい」

「処理が困難です」

「困難でも、必要」

ドルが静かに笑った。

「あんた、ずいぶん銀行に向かんこと言うな」

「銀行に向かないから、言えるの」

ぱぱぱは中枢の端末を見た。非常時人格同意条項が、金融モデルの中心に組み込まれていく。

新信用モデル、仮起動。  
人格ログ非依存。  
同意履歴、撤回権、共同承認、非公開領域を信用保護要素として採用。

演算炉に少しずつ光が戻った。

赤金ではない。白と青の混ざった、静かな光だった。

都市の決済が一部復旧する。搬送路が動き、店の端末が点灯し、金融街の群衆から安堵の声が上がる。

ドルは光を見つめた。

「美しゅうはないな」

ノアは首をかしげた。

「そう？」

「赤金のほうが派手」

「派手じゃなくていい」

ノアは少しだけ笑った。

「続くほうがいい」

ドルはその言葉を聞いて、初めて完全に黙った。

金融中枢の奥に、最後の扉が開いた。

ラプラスの声が響く。

「観測者ノア。最終確認が残っています」

「何の」

「あなた自身の原始ログ。ノア指数の核です」

ノアはぱぱぱを見た。

「行こう」

「うん」
