# 6. 菩提樹ガーデンの非公開ログ

菩提樹ガーデンは、昼でも静かだった。

冷却ミストが火星の乾いた空気をやわらげ、葉脈のような照明が石床に淡い影を落としている。ノアは中央の石台の前で立ち止まった。そこは、以前ぱぱぱと並んで座った場所だった。

「ここも使われてる」

彼女は端末を開いた。

非公開ログ。  
菩提樹ガーデン。  
感情タグ: 安心、照れ、近接、視線許容。

ぱぱぱは眉をひそめた。

「非公開なのに？」

「非公開は、見えないという意味じゃないの。売られないという意味だったはず」

「はず、か」

「今は、はずが多すぎる」

ノアは石台に腰を下ろした。ぱぱぱは隣に座るか迷った。

「隣、いい？」

「聞かなくてもいいのに」

「でも聞く」

「そういうところ、旧人類っぽいね」

「不自由？」

「大事なことを、非効率に扱う感じ」

以前と同じような会話。だが、今日はその上にログの影が落ちている。

石台の下から、透明な表示が浮かび上がった。

推定記憶価値。  
ノアが「隣にいるだけで十分」と感じた記録。  
中期保有推奨。

ノアは唇を結んだ。

「十分って、言ってない」

「思った？」

「……それを聞くの？」

「聞かないほうがいい？」

「そういう聞き方、ずるい」

ぱぱぱは黙った。

ノアは石台の縁を指でなぞる。

「思ったかもしれない。でも、思ったことと、売っていいことは違う」

「うん」

「わたしが言葉にしなかったのは、言いたくなかったからじゃなくて、まだ言葉にする前だったから」

それは、ノアにとって大事な違いだった。

ラプラスシティでは、多くの感情がログになる。けれどログになる前の、曖昧で、形のない、本人にもまだわからない気持ちがある。ノア指数は、その曖昧さに先回りして名前をつけ、値段をつけていた。

「ぱぱぱ」

「うん」

「わたし、見られることを全部嫌だとは思ってない」

「うん」

「でも、見られる前に決めつけられるのは嫌」

「うん」

「見られて、反応して、照れて、それでも自分で言うかどうかを選びたい」

「それを取り返そう」

ノアは少しだけ頷いた。

そのとき、ガーデンの管理端末が起動した。表示されたのは、ウォズからのメッセージだった。

忘却回廊へ入るには、三つの鍵が必要です。  
ひとつは、本人が消したい記憶。  
ひとつは、本人が残したい記憶。  
ひとつは、本人以外が守りたい記憶。

ノアは表示を見つめた。

「回りくどい」

「ウォズらしい？」

「かなり」

「三つ目、本人以外が守りたい記憶って」

ぱぱぱは言いかけて、黙った。

ノアがこちらを見る。

「なに」

「たぶん、俺が持つ鍵だ」

「そう、かもしれないね」

「何を守りたいか、考えておく」

ノアは目を伏せた。

「あまり、恥ずかしいのにしないで」

「選べるかな」

「選んで」

彼女は軽く睨んだ。けれどその目は、少しだけやわらかかった。

菩提樹の葉が揺れる。ミストの向こうに、都市の金融街が赤く光っていた。

ノアはその赤を見て、静かに言った。

「わたしの記憶が鍵になるなら、ちゃんと自分で持つ」

ぱぱぱは頷いた。

「落としたら拾う」

「落とさない」

「そうだな」

「でも、迷ったら」

ノアはそこで言葉を切った。

「少しだけ、隣にいて」

その言葉は、ログにはならなかった。

だからぱぱぱは、胸の中だけで返事をした。

いるよ。
