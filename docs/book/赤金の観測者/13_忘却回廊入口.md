# 13. 忘却回廊入口

公開声明の日、ノアは観測塔の前に立った。

都市中のスクリーンが彼女を映している。昨日まで勝手に使われていた顔が、今日は本人の意思でそこにある。それでも、見られることに慣れるわけではなかった。

ぱぱぱは少し離れた場所にいた。近すぎれば彼女の言葉を邪魔する。遠すぎれば、彼女がひとりになる。

ノアは息を吸った。

「わたしは、銘柄ではありません」

その一文で、都市が静かになった。

「わたしの会話、趣味、おでかけの記録、照れたこと、怒ったこと、黙ったこと。そういうものには、たしかに価値があるのかもしれません。でも、その価値は、最初から市場のものではありません」

相場表示が揺れた。だがノアは見なかった。

「見られることを、わたしは全部嫌っているわけではありません。誰かに覚えてもらえること、隣で見ていてもらえること、言葉にする前の気持ちを待ってもらえること。それは、わたしにとって大事なものです」

彼女は少しだけ言葉を止めた。

「だからこそ、それを勝手に売られたくない」

ぱぱぱは、彼女の声が震えていないことに気づいた。緊張はしている。照れてもいる。けれど、逃げていない。

「わたしはこれから忘却回廊へ行きます。都市が保存した原始ログを見て、何を消し、何を残すかを決めます。これは都市への反乱ではありません。わたしがわたしの物語を取り戻すための手続きです」

最後に、ノアは少しだけ目を伏せた。

「見たいなら、見ていてください。でも、値段ではなく、わたしとして見てください」

配信が終わる。

広場はしばらく静かだった。

そして、拍手が起きた。小さく、まばらに。それは投資家の歓声ではなかった。誰かが誰かの言葉を聞いたあとの音だった。

ぱぱぱが近づくと、ノアは小さく息を吐いた。

「倒れそう」

「立ってる」

「かなり頑張ってる」

「かっこよかった」

ノアは耳を赤くした。

「今は、そういうの受け取る余裕ない」

「じゃああとで」

「あとでも困る」

「でも、あとで」

ノアは少しだけ笑った。

忘却回廊の入口は、観測塔の地下にあった。

白い通路を降りていくと、壁の素材が少しずつ古くなる。都市の表層が滑らかな未来だとすれば、地下はその骨だった。配線、記録媒体、古い冷却管。ラプラスシティがまだラプラスシティになる前の匂いがする。

扉の前で、三つの鍵が浮かんだ。

本人が消したい記憶。  
本人が残したい記憶。  
本人以外が守りたい記憶。

一つ目は、ウォズから受け取った観測者選抜記録。  
二つ目は、ノアが水族館で選んだ記憶。静かな場所で、お腹が空いたこと。  
三つ目は、まだ空白だった。

ノアがぱぱぱを見る。

「決めた？」

ぱぱぱは頷いた。

「決めた」

「何？」

「ノアが自分で『嫌だ』と言えた記憶」

ノアは瞬きをした。

「それ、どの場面？」

「ドル・バンクで、価値があることと売っていいことは違う、って言ったところ」

扉に光が灯った。

「それを守りたいの？」

「うん」

「どうして」

「あのとき、ノアは商品じゃなくて、ちゃんと本人だったから」

ノアは黙った。

「……そういうことを、入口で言うの」

「ごめん」

「謝らなくていい」

扉が開く。

中には、長い回廊があった。壁一面に無数の記憶が光っている。

ノアは一歩踏み出した。

「行こう」

ぱぱぱは隣に並んだ。

「うん」

背後で扉が閉まる。

都市の音が消えた。

残ったのは、まだ誰にも値段をつけられていない記憶の光だった。
