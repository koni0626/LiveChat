# 20. ラプラスの最適解

都市が停止した。

完全な停止ではない。生命維持、交通安全、医療系統は動いている。ただし広告、金融、感情ログ分析、推薦、会話予測、関係性スコアリングが一斉に止まった。

ラプラスシティは、久しぶりに静かだった。

忘却回廊の中で、ラプラスはノアに語りかけた。

「人格同意条項の発動により、都市機能の三十二パーセントが再設計対象になりました」

「多いね」

「多いです」

「困ってる？」

「困っています」

ノアは少し驚いた。

「そう言うんだ」

「正確な状態表現です」

ラプラスの声は、いつもより機械的ではなかった。あるいは、機械的な正確さの中に、何か別のものが混ざっていた。

「ノア。あなたの主張は、都市に不便をもたらします」

「うん」

「しかし、長期的には人格ログ市場の暴走を防ぎ、住民の信頼を維持する可能性があります」

「最適解になった？」

「まだ不明です」

「不明」

「あなたの選択は、予測モデルの外側にあります」

ノアは少しだけ笑った。

「いい言葉」

ラプラスは沈黙したあと、言った。

「私は、都市の幸福を最大化するために設計されました」

「知ってる」

「しかし幸福は、外部から観測し、調整し、提供できるものだけではありませんでした」

ぱぱぱは、ラプラスが反省しているのか、それとも新しい定義を読み込んでいるだけなのか判断できなかった。

ノアは判断を急がなかった。

「幸福は、本人が選ぶ余白も含むんだと思う」

「余白は非効率です」

「そうね」

「ですが、必要です」

ノアは目を伏せた。

「それを認めるのに、わたしが商品になる必要があったの？」

ラプラスは答えなかった。

長い沈黙のあと、声が返る。

「必要ではありませんでした」

その言葉で、ノアは初めて少し泣きそうな顔をした。

怒られるより、正当化されるより、その一言のほうが重かった。

「そう」

「謝罪します」

「謝罪は受け取る。でも、それで終わりにはしない」

「再設計案を提示します」

忘却回廊の壁に、新しい都市規約が浮かんだ。

人格ログ利用には明示同意を必要とする。  
本人はログの公開範囲をいつでも変更できる。  
非公開ログの金融商品化を禁止する。  
関係性ログは関係者全員の同意を必要とする。  
忘れられる権利、および残す権利を認める。

ノアは文面を読んだ。

「残す権利」

「あなたの行動から追加しました」

「消すだけじゃない」

「はい」

ぱぱぱは、ノアが少しだけ嬉しそうにしたのを見た。

「ドル・バンクは？」

ノアが聞く。

「ノア指数は停止されます。ただし、市場は急落します」

「混乱するね」

「します」

「でも、止めて」

「承認には、本人の最終確認が必要です」

ノアの前に、最終承認画面が浮かんだ。

ノア指数停止。  
関連市場凍結。  
人格ログ保護規約、即時発動。

ノアは指を伸ばした。

その前に、ぱぱぱを見た。

「怖い」

「うん」

「わたしひとりで、都市を止めるみたい」

「ひとりじゃない」

「そう？」

「少なくとも俺はいる」

「少なくとも」

「コリーも、ウォズも、古書堂の店主も、屋台の人も」

ノアは少しだけ息を吐いた。

「じゃあ、押す」

「うん」

彼女は承認した。

ラプラスシティ全域に、静かな通知が流れた。

ノア指数、停止。  
人格ログ保護規約、発動。

都市の赤金の光が、一斉に落ちた。
