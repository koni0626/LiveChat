# 11. ウォズの回りくどい告発

ウォズは、観測塔の古い記録室にいた。

黒い外套をまとい、何世代も前の端末を前にしている。彼の周囲だけ、ラプラスシティの時間が少し古く見えた。

「ようやく来ましたか」

ノアは腕を組んだ。

「あなた、ずっと見てたでしょう」

「見ていました」

「助けるなら、もっと早く助けて」

「早すぎる助けは、本人の選択を奪います」

「便利な言い訳ね」

ウォズは否定しなかった。

「ええ。便利です。ですが、今回はそれだけではありません」

彼は古い端末を操作した。空中に、ラプラスシティの初期設計図が浮かぶ。都市の中心には観測塔、地下には巨大な記憶基盤、そしてその下に忘却回廊が描かれていた。

「メモリー・クレジットは、ドル・バンクだけの発明ではありません。原型は都市の初期設計にあります」

ノアの表情が固くなる。

「ラプラスが作った？」

「ラプラスは都市を維持するため、住民の感情流通を読む必要がありました。最初は医療、治安、孤独の予防。善意の仕組みでした」

「善意から始まる厄介なもの、多すぎない？」

「歴史とは、だいたいそういうものです」

ウォズは次の記録を開いた。

そこには、若いころのノアのデータがあった。観測者候補としての適性検査。感情反応の豊かさ。否定と本音の差。見られることへの矛盾した反応。

ノアは息をのんだ。

「これ、いつの」

「あなたが観測者になる前です」

「わたし、知らない」

「知らされていません」

ぱぱぱはウォズを睨んだ。

「それを今まで黙ってたのか」

「告発には、聞く側の準備が必要です」

「回りくどい」

「そういう役目です」

ノアは記録を見つめていた。

そこには、彼女が観測者に選ばれた理由が書かれていた。

他者の感情変化に敏感。  
自身も感情反応が豊か。  
観測する側と観測される側の境界が揺れやすい。  
都市の感情流通を読む媒介として適性高。

「わたしは、最初からこういうふうに見られていたの」

「はい」

「観測者として？」

「そして、観測対象として」

ノアは黙った。

ぱぱぱは何か言おうとしたが、言葉が出なかった。慰めでは足りない。怒りでも足りない。

ノアはゆっくりと端末に触れた。

「この記録は、消したい」

ウォズが頷く。

「一つ目の鍵になります。本人が消したい記憶」

「でも、これを消したら」

「あなたが観測者に選ばれた理由の一部が消える」

ノアは目を閉じた。

「消したいのに、消すのが怖い」

「それで正常です」

ウォズは静かに言った。

「記憶は、傷であると同時に輪郭です。すべて消せば楽になるとは限りません」

ノアはぱぱぱを見た。

「どう思う？」

「俺が決めることじゃない」

「そう言うと思った」

「でも、隣にはいる」

ノアは少しだけ笑った。

「それも、そう言うと思った」

ウォズが小さなデータ片を差し出した。

「消したい記憶の鍵です。忘却回廊で使えます。ただし、使うかどうかは内部で決めなさい」

ノアはそれを受け取った。

「ウォズ」

「はい」

「あなたは、わたしを観測対象として見てる？」

ウォズは少し考えた。

「見ています」

ノアの目が細くなる。

「ですが、商品としては見ていません」

「その違いを、ちゃんと説明できる？」

「観測は、相手が相手であることを保存する行為です。商品化は、相手を使いやすい形に削る行為です」

ノアはしばらく黙っていた。

「その説明は、少し好き」

「光栄です」

「でも、回りくどい」

「仕様です」

ぱぱぱは少し笑った。ノアも少しだけ笑った。

その笑いは、記録室の古い端末にだけ残った。
