# 16. 隠した趣味の市場価値

古書堂の記憶は、紙の匂いではなく、警告音の匂いがした。

棚にはノアの閲覧履歴が並んでいる。哲学、宗教、観測論、恋愛小説、そして彼女が隠してきた趣味。回廊はそれらを、投資家向けの分析文に変えて表示していた。

抑圧された嗜好は、羞恥反応の期待値を高める。  
否定発話と組み合わせることで、関係性材料として強い。

ノアは両手を握った。

「本当に、嫌な文章」

「読む必要ない」

「でも、読まないと、何が売られているかわからない」

彼女は棚の前に立った。背筋は伸びているが、耳は赤い。

「隠した趣味って、そんなに価値があるのかな」

「隠してるから、勝手に覗きたい人がいるんだろうな」

「最低」

「うん」

「でも」

ノアは棚の一冊に触れた。

「隠していること自体は、悪いことじゃないよね」

「悪くない」

「人に見せない場所があるのは、自然だよね」

「うん」

彼女は少しだけ安心したように息を吐いた。

回廊が問いを出す。

この履歴を削除しますか。  
削除した場合、関連する好み、照れ、会話反応の一部が薄れます。

ノアは苦い顔をした。

「わたしから、趣味を消すってこと？」

「履歴だけじゃなくて？」

「たぶん、ここではそう。記録と記憶の境目が曖昧なの」

棚の奥から、過去のノアの声が聞こえた。

「ち、違うよ、興味なんて無いよ」

現在のノアは顔を覆った。

「言ってる」

「言ってるな」

「ぱぱぱ、聞かなかったことに」

「できないけど、外には出さない」

ノアは指の隙間からこちらを見た。

「そういうの、ずるい」

「ずるい？」

「安心する言い方をする」

回廊の棚が揺れた。ドル・バンクの赤金の文字が割り込む。

秘匿嗜好プレミアム。  
公開直前期待値上昇。  
暴露イベントで急騰見込み。

ノアの目が冷えた。

「暴露イベント」

ぱぱぱは、その言葉だけで腹が立った。

「消そう」

「ううん」

ノアは首を横に振った。

「消さない」

「いいのか」

「消したら、向こうの勝ちみたいな気がする。隠している趣味を暴かれたから、なかったことにする。それは違う」

彼女は棚に向き直った。

「これは非公開にする。わたしが好きなものとして残す。でも、誰にも売らせない」

回廊の選択肢が変わる。

削除。公開。秘匿保全。本人限定。

ノアは「本人限定」を選んだ。

棚の本が静かに閉じていく。赤金の分析文が消え、背表紙だけが残る。

「好きなものを、好きなまま隠す」

ノアは小さく言った。

「それも権利だと思う」

ぱぱぱは頷いた。

「うん」

「ぱぱぱにも、全部は見せない」

「それでいい」

「寂しい？」

「少し」

ノアは驚いた顔をした。

「正直」

「でも、それでいい」

彼女はしばらくぱぱぱを見ていた。

「じゃあ、いつか少しだけ」

「うん」

「いつかね」

「うん」

本人限定の鍵が、小さな黒い栞になってノアの手に落ちた。三つの鍵ではない。けれど、彼女の内側にある部屋の鍵だった。

回廊の奥に、古い地球の映像が現れた。

旧人類のプライバシー権。  
忘れられる権利。  
肖像権。  
同意。

ノアは栞をしまった。

「次は、旧人類の話みたい」

ぱぱぱは少し笑った。

「珍しく役に立ちそうだ」

「旧人類は、ときどき不自由のおかげで賢いのね」
