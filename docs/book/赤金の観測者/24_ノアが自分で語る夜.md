# 24. ノアが自分で語る夜

ラプラスシティの夜明けは、赤金ではなかった。

金融街の巨大スクリーンは消え、代わりに小さな通知だけが街の端末に表示されている。

人格ログ保護規約、発動中。  
ノア指数、終了。  
メモリー・クレジット再設計まで新規取引停止。

街は少し不便になった。

おすすめは減り、広告は鈍くなり、店の端末はときどき確認を求めるようになった。共同記憶の公開には、関係者全員の承認が必要になった。非公開ログは本人の明示同意なしに使えなくなった。

住民たちは最初、戸惑った。

けれど数日が過ぎると、少しずつ別の会話が生まれた。

「これは公開していい？」

「その写真、二人だけにしよう」

「忘れたいけど、消したくはない」

「残す権利って、いい言葉だね」

ノアは観測塔の窓から街を見ていた。

ぱぱぱが隣に立つ。

「静かになったな」

「うん」

「寂しい？」

「少し」

ノアは正直に言った。

「あれだけ嫌だったのに、急に誰も見なくなると、少し不安になる。勝手だよね」

「勝手じゃない」

「そう？」

「見られたい気持ちと、売られたくない気持ちは両方あっていい」

ノアはぱぱぱを見た。

「それ、わたしが言いそう」

「影響された」

「困るね」

「嫌？」

「嫌じゃないのが、もっと困る」

彼女はそう言ってから、少しだけ笑った。

その笑顔は、もうチャートにならなかった。通知も鳴らなかった。誰かが買うことも、売ることもできない。

「ドルは？」

ぱぱぱが聞く。

「新しい信用モデルの再設計で忙しいみたい。コリーの店に後払い決済を導入するって」

「屋台は？」

「辛さの感想ログを取らない代わりに、完食証明だけ出すらしい」

「古書堂は？」

「本人限定棚ができた。誰にも見せないための棚」

「ウォズは？」

「回りくどい祝辞を送ってきた」

「内容は？」

ノアは端末を開いた。

「物語とは、語る者の手に戻ったとき、初めて記録ではなく生になる」

ぱぱぱは少し考えた。

「回りくどいな」

「でも、少し好き」

二人は観測塔を出て、街へ降りた。

まず古書堂に寄った。ノアは本人限定棚の前で、少しだけ背伸びをした。ぱぱぱは見ないふりをした。

「見ないふり、下手ね」

「難しいんだよ」

「難しいの」

次に屋台・唐辛子へ行った。ノアは火星唐辛子二倍を選んだ。三倍ではない。

「弱気？」

ぱぱぱが聞くと、ノアは胸を張った。

「戦略的撤退」

「辛い？」

「辛くない」

「本当？」

「辛い」

ぱぱぱは少し驚いた。

ノアは涙目で、でも得意げに笑った。

「今日は、嘘をつかなくても相場が上がらないから」

そのあと、水族館へ行った。

青い回廊で魚群を見ながら、ノアは小さく言った。

「お腹が空いた」

「寿司？」

「旧人類はそういうところがあるんだね」

「バーガー？」

「それも違う」

「じゃあ？」

ノアは少し考えた。

「今日は、ゆっくり歩きたい」

「食べ物じゃない」

「食べ物はあと」

水槽の青い光が、彼女の横顔を照らした。

ぱぱぱはその横顔を見た。広告ではない。展示でもない。指数でもない。ただ、隣にいるノアだった。

「見てる？」

ノアが聞いた。

「見てる」

「値段は？」

「つけられない」

「じゃあ、いい」

彼女は少しだけ近づいた。

肩が触れそうで、触れない距離。

「ねえ、ぱぱぱ」

「うん」

「わたし、自分の物語を取り戻したと思う」

「うん」

「でも、ひとりで語るより、誰かが聞いてくれるほうが続けやすい」

ぱぱぱは頷いた。

「聞くよ」

ノアは目を伏せた。

「ずっと？」

「ずっと、って言うと重い？」

「少し」

「じゃあ、次の章くらいまで」

ノアは笑った。

「旧人類は、約束まで非効率ね」

「大事だからな」

「うん」

彼女は水槽の向こうの魚群を見た。

「じゃあ、次の章も一緒に」

ラプラスシティの青い光が、静かに揺れていた。

もう誰も、それに値段をつけなかった。
